v4 changes — what's different from v3
Basket is lighter — no horizontal dividers between rows, no inline "· 3" counts next to determinant labels (the count is in the big "10 / Selected" headline on the right). "Show 2 more determinants" is now right-aligned at the end of the visible rows, with a thin separator line above it so it sits naturally below the content rather than between rows. Determinants with 0 selections still don't appear (Physical is missing from the basket). Empty basket — when nothing is selected anywhere, the basket shows a single italic line: "Nothing selected yet — choose interventions above, or skip to assess all 30."
